# 💳 Credit Card Financial Analytics Dashboard using Microsoft Power BI

## 📌 Project Overview

This project is an interactive Financial Analytics Dashboard developed using Microsoft Power BI. It provides insights into credit card customer data, account trends, credit limits, card distribution, and financial performance using interactive visualizations.

The dashboard helps financial institutions understand customer portfolios, monitor credit utilization, and analyze card issuance trends.

---

## 🎯 Objectives

- Analyze customer and credit card data.
- Monitor account growth over time.
- Compare card brands and card types.
- Evaluate credit limits across customers.
- Explore card issuance trends.
- Enable interactive filtering using slicers.

## 📊 Dashboard

### Executive Overview

#### Features

- Total Customers
- Total Cards
- Average Credit Limit
- Maximum Credit Limit

- Card Type Distribution
- Accounts Opened Over Time
- Credit Limit by Card Brand
- Card Issuance by Brand




---

## 📈 Key Insights

- Compare Credit vs Debit (Prepaid) card distribution.
- Analyze customer credit limits across different card brands.
- Monitor account creation trends over time.
- Identify brands with the highest number of issued cards.
- Track overall customer portfolio performance.
